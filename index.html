How to use the weekly tracker
A quick guide to getting the most out of your tracker.
The three columns

Each working week is organised into three columns. Add tasks to whichever column fits best — you can always move them later.

● In progress Tasks you are actively working on right now. ● Planned Tasks you intend to start this week but haven't yet. ● Blocked Tasks you cannot progress because something or someone is in the way. Use the notes feature to explain what the blocker is.
Task buttons
✓ Mark a task as completed. It moves to the Completed section at the bottom. ✗ Mark a task as cancelled — for work that has been dropped or is no longer needed, often for reasons outside your control. → Move a task to a different column. For example, move a planned task to Blocked if it hits a problem, or move a blocked task back to In progress once the blocker is resolved. ◇ Mark a task as ongoing. Ongoing tasks automatically carry over to the next week without any manual action. The diamond turns solid (◆) when active. Use this for long-running work that spans several weeks or months. +▲ Add or edit a note on a task. Use this to give context — such as what is causing a blocker, where a task is up to, or any relevant detail for your manager. ↩ Restore a completed or cancelled task back to its original column if you marked it by mistake. ✕ Permanently delete a task. This cannot be undone.
Tags

Tags appear automatically on tasks to show their history at a glance.

carried This task was carried over from a previous week.
ongoing This task will automatically carry over each week until marked complete or cancelled.
Navigating weeks

Use the arrows at the top to move between weeks. Each week's tasks are stored separately, so you can look back at previous weeks at any time.

If you have incomplete tasks from a previous week, a blue banner will appear at the top offering to carry them all over to the current week in one click.

Manager update

The manager update preview at the bottom of the page builds automatically as you add tasks. When you're ready, click Copy update to copy the formatted text to your clipboard and paste it into an email, Teams message, or Slack.

Saving your data

Your data saves automatically to your browser's local storage every time you make a change. As long as you use the same browser on the same device, your tasks will be there when you return.

Use Export JSON to download a backup of all your data at any time. Use Import JSON to restore from a backup or move your data to a different device.

Import JSON
Paste a previously exported JSON file to restore your data. This will overwrite your current data.
